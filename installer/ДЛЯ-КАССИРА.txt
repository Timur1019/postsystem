================================================================================
  AURENT CASHIER — установка на компьютер кассира
================================================================================

  Сервер уже работает в облаке. На кассе нужно только это приложение.

--------------------------------------------------------------------------------
  MAC (MacBook / iMac)
--------------------------------------------------------------------------------

  1. Откройте папку mac/
  2. Дважды щёлкните файл  Aurent Cashier-1.0.0.dmg
  3. Перетащите «Aurent Cashier» в папку «Программы»
  4. Запустите «Aurent Cashier» из Программ

  При первом запуске:
    • Адрес сервера: 111.88.132.126  (уже подставлен)
    • Порт:           8080
    • Нажмите «Сохранить и подключиться»

  Вход:
    • Логин и пароль выдаёт администратор магазина
    • (по умолчанию на сервере: admin / password — смените!)

--------------------------------------------------------------------------------
  WINDOWS
--------------------------------------------------------------------------------

  1. Откройте папку windows/
  2. Запустите  Aurent Cashier Setup 1.0.0.exe
  3. «Далее» → установить → ярлык на рабочем столе
  4. Откройте «Aurent Cashier»

  При первом запуске укажите:
    • Сервер: 111.88.132.126
    • Порт:   8080

  Если Windows спрашивает «Разрешить?» — нажмите «Да».

--------------------------------------------------------------------------------
  ПРОВЕРКА СВЯЗИ С СЕРВЕРОМ
--------------------------------------------------------------------------------

  В браузере на кассе откройте:

    http://111.88.132.126:8080/api/v1/actuator/health

  Должно показать:  {"status":"UP"}

  Если не открывается — проверьте интернет и обратитесь к администратору.

--------------------------------------------------------------------------------
  СМЕНИТЬ АДРЕС СЕРВЕРА
--------------------------------------------------------------------------------

  Mac:     дважды щёлкните «Настройка-сервера-Mac.command» в этой папке
  Windows: «Настройка-сервера-Windows.bat»

  Или в программе: меню Aurent → «Настройка сервера…»
